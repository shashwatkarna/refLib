# Research Paper Formatting AI Tool

This tool is a Python-based utility to automatically process and reformat an academic research paper based on standard Indian academic guidelines. 

## Features
- Preserves the content of `.pdf` and `.docx` formatted papers.
- Automatically adjusts document formatting:
  - 1-inch margins
  - Times New Roman, 12pt
  - Double Spacing
  - Indented references and paragraphs
  - Standardized heading structures

## Installation
1. Install Python 3.8+ 
2. Install the necessary packages via `pip`:
   ```bash
   pip install -r requirements.txt
   ```
   
## Usage
1. Run the application:
   ```bash
   python app.py
   ```
2. A graphical user interface will open.
3. Click **Select File** and locate your unformatted research paper `.docx` or `.pdf`.
4. Choose your citation style (e.g., APA/MLA).
5. Click **Format Document** and choose where to save your formatted output!
